# 📖 Overview

The film industry generates massive amounts of structured data spanning movie performance, ratings, budgets, earnings, genres, and audience engagement.

This project demonstrates how SQL and SQLite can be used to query relational databases, combine information across multiple tables, and generate business insights from movie data.

Using Python and Pandas, SQL queries are executed directly against a SQLite database to answer real-world analytical questions.

---

# 🎯 Business Objective

Explore the IMDb dataset to answer questions such as:

- Which movies received the highest audience ratings?
- Which films generated the highest profits?
- Which genres dominate the industry?
- How do budgets vary across genres?
- Which years produced the highest-grossing movies?
- What relationship exists between investment and earnings?
